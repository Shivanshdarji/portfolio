## ✨ Features

### 🎨 Design & Aesthetics
- **Dark Futuristic Theme** — Deep space palette with purple/cyan neon accents
- **Neural Network Background** — Interactive particle canvas that responds to mouse movement
- **Custom Cursor** — Glowing dot with a trailing ring that expands on interactive elements
- **Glassmorphism Cards** — Frosted glass effect on project and achievement cards
- **Smooth Scroll Animations** — Elements reveal on scroll with staggered delays

### ⚡ Interactive Effects
- **Typewriter Terminal** — macOS-style terminal cycling through AI/ML commands
- **3D Tilt on Project Cards** — Cards tilt in 3D space following cursor position
- **Animated Skill Bars** — Gradient progress bars with glowing cyan endpoints
- **Counter Animation** — Stats count up from zero when scrolled into view
- **Parallax Scrolling** — Hero content fades and shifts as you scroll
- **Easter Egg** — Press `↑ ↑ ↓ ↓` for a color shift surprise!
- **Tab Title Change** — Shows "Come back! 👋" when you leave the page

### 📱 Responsive
- Fully responsive across desktop, tablet, and mobile
- Mobile hamburger menu with smooth slide-in animation
- Optimized touch interactions for mobile devices

## 🚀 Getting Started

### View Locally
```bash
# Clone the repository
git clone https://github.com/Shivanshdarji/portfolio.git

# Navigate to the project
cd portfolio

# Open in browser (or use Live Server)
# Option 1: Simply open index.html in your browser
# Option 2: Use Python's built-in server
python -m http.server 8080

# Visit http://localhost:8080
```

### Deploy on GitHub Pages
1. Go to your repo → **Settings** → **Pages**
2. Under **Source**, select **Deploy from a branch**
3. Choose `main` branch and `/ (root)` folder
4. Click **Save** — your site will be live at `https://shivanshdarji.github.io/portfolio`
